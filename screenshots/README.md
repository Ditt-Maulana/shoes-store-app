# Screenshots

Folder ini berisi screenshot dari aplikasi Flutter Shoe Store.

## 📸 Nama File yang Dibutuhkan:

1. `login.png` - Screenshot halaman login
2. `home.png` - Screenshot halaman home dengan produk
3. `profile.png` - Screenshot halaman profile
4. `search.png` - Screenshot halaman search
5. `notification.png` - Screenshot halaman notifications
6. `edit-profile.png` - Screenshot halaman edit profile

## 📐 Rekomendasi Ukuran:

- **Width**: 300-400px
- **Height**: 600-800px (portrait/mobile view)
- **Format**: PNG atau JPG

## 🎯 Cara Mengambil Screenshot:

1. Jalankan aplikasi: `flutter run -d chrome`
2. Buka DevTools (F12) → Toggle Device Toolbar (Ctrl+Shift+M)
3. Pilih device: iPhone 12 Pro atau Pixel 5
4. Screenshot setiap halaman
5. Save dengan nama file sesuai di atas
6. Copy ke folder ini
